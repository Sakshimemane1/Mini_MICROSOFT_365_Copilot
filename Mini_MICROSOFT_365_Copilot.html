The Problem I Noticed

While exploring mobile productivity apps like Word, Excel, and PowerPoint, I realized that finding relevant documents quickly is often difficult. Users waste time searching across apps, and current AI-powered suggestions are not always intuitive. I wanted to explore how context-aware search could make productivity seamless.

My Solution

I personally built a Mini Microsoft 365 Copilot Web Demo to simulate an AI-powered search experience for mobile. I handled everything end-to-end—from design and layout to coding the interactive search and deploying the live demo. This project reflects my initiative, problem-solving skills, and ability to think like a product engineer.

Mini Copilot Demo

The demo features AI-style search suggestions across multiple document types and a mobile-first responsive layout that resembles an iOS app. The datasets and search logic were curated and coded by me to simulate context-aware results.
